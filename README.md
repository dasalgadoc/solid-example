## Environment Setup

### Needed tools

1. Java 11
2. Maven

### Application execution

1. Make sure to download all Needed tools
2. Clone the repository
```
  git clone https://github.com/dasalgadoc/solid-example.git
```
3. Build up maven project
```
  mvn dependency:resolve
```
